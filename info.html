HOBI

ANIMASI

Lebih tepatnya yang saya buat adalah animasi 2D rigging, di mana per bagian (seperti badan, tangan, kaki) saya jadikan satu, kemudian bisa digerakkan bebas tanpa perlu menggambar ulang. Untuk melakukannya saya menggunakan aplikasi Macromedia Flash. Aplikasi ini juga saya gunakan untuk tugas akhir karena ringan dan pas untuk laptop dengan spek kentang.

EDIT GAMBAR

Untuk edit cepat saya lebih terbiasa menggunakan Affinity Designer, sedangkan untuk aplikasi yang lebih populer seperti Adobe Photoshop atau Canva level saya sama dengan orang yang baru belajar.

FOTOGRAFI / DOKUMENTASI

Saya terkadang membantu kegiatan gereja ketika acara besar seperti HUT gereja, pembaptisan, natal dan juga dokumentasi kegiatan lainnya. Alat yang saya gunakan hanya handphone dan kamera DSLR (masih pemula).

MECHANICAL KEYBOARD

Hanyalah sebuah benda mati yang bisa dipencet-pencet dan tujuannya adalah sekedar estetika setup atau untuk bekerja dan bermain game. Merawat dan memodifikasinya sesuai yang saya inginkanlah yang menjadikannya salah satu hobi saya.

BIKIN GAME

Baru-baru ini saya mencoba untuk membuat game yang hanya bermodalkan javascript di HTML saja. Game yang saya buat baru saja dalam progress coba-coba dan mungkin akan hiatus dalam waktu dekat ketika logika game sudah saya kuasai tapi belum dapat ide untuk mau diteruskan jadi apa.
