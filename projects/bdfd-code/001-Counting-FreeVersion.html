Basic "Clean" Counting

It's for free version that uses 1-9 as triggers. For basic explanation you can go to here

Before we start

First: You will need 2 variables only.

If you know how to use JSON or Text Spliting, that would be better because you can only use 1 variable.

But I'll explain the basic stuff, here :)

Example Variables

1. Name: countingUser and Value: none

2. Name: countingNumber1 and Value: 1

Why not 0? Because you'll need 0-9 triggers that way and 0 is just gonna be typed only once anyway.

Trigger: 1

$var[number;1$message]
$onlyIf[$cropText[$getVar[countingNumber];1;]==1;]
$if[$authorID!=$getVar[countingUser]]
	$if[$var[number]==$getVar[countingNumber]]
		$addCmdReactions[âœ…]
		$setVar[countingNumber;$calculate[$getVar[countingNumber]+1]]
		$setVar[countingUser;$authorID]
	$elseif[$var[number]!=$getVar[countingNumber]]
		$deletecommand
		You input the wrong number, my brother!
		$deleteIn[5s]
	$endif
$elseif[$authorID==$getVar[countingUser]]
	$deletecommand
	You can't count anymore, just one at a time, my brother!
	$deleteIn[5s]
$endif

So basically what I did here is to know what the trigger we type using a temporary variable ($var). Since normal $message or $message[1] will ignore the trigger.

Then, I used $onlyIf function to check if the first character of current value of $getVar is the same as trigger or no and if it's not same then give no response.
